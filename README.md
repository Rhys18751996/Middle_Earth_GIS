# Middle_Earth_GIS
Building a digital Middle-earth GIS: terrain, settlements, roads, regions, history, and lore represented as interconnected geospatial data.

Middle_Earth_GIS is an experimental GIS-inspired platform built in Unity that aims to represent the world of Middle-earth as structured geospatial data.
The project combines terrain visualization, map layers, settlements, roads, regions, lore locations, historical events, and community-contributed datasets into a searchable and extensible world model.
The long-term goal is to create a platform that functions similarly to a real-world GIS system while remaining faithful to Tolkien's legendarium.
